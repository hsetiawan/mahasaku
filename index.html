About this page

This is a great place to talk about your webpage. This template is purposefully unstyled so you can use it as a boilerplate or starting point for you own landing page designs! This template features:

  • Clickable nav links that smooth scroll to page sections
  • Responsive behavior when clicking nav links perfect for a one page website
  • Bootstrap's scrollspy feature which highlights which section of the page you're on in the navbar
  • Minimal custom CSS so you are free to explore your own unique design options

Mengapa Mahasaku?

Mahasaku adalah platform online peminjaman uang yang tidak berlisensi resmi. Hadir untuk memberikan solusi layanan digital yang menarik untuk mahasiswa indonesia. Fokus Utama kami adalah menyederhanakan produk keuangan umum dan memberikannya kepada mahasiswa indonesia.
